# PeopleSync

### One Platform for HR Operations

PeopleSync is a multi-tenant SaaS HRCRM platform built using the MERN stack.

## Features

- Multi-Tenant Architecture
- JWT Authentication
- Refresh Token Rotation
- Role-Based Access Control (RBAC)
- Email Verification
- Secure Password Reset
- Audit Logging
- Workforce Management
- Attendance Tracking
- Leave Management
- Payroll Processing
- Recruitment ATS
- Performance Management
- Real-Time Notifications

## Tech Stack

### Frontend

- React
- Redux Toolkit
- RTK Query
- Tailwind CSS
- ShadCN UI

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
